hexo 설정 파일

Configuation

_config.yml 파일을 수정하면 사이트 설정을 바꿀 수 있습니다.

1. Site

속성 설명
title 웹사이트의 타이틀
subtitle 웹사이트의 서브타이틀
description 웹사이트 설명
author 홈페이지 제작자 이름
language 웹사이트 언어를 나타내는 두글자 코드. 링크
timezone 웹사이트 타임존, 컴퓨터의 설정값을 기본값으로 사용함.

2. URL

속성 설명
url 웹사이트 url
root 웹사이트 root 디렉토리
permalink 글의 퍼머링크 포맷
peramlink_defaults 각 세그먼트 퍼머링크의 기본값

만약, 웹사이트가 하위 디렉토리에 있다면 아래 예제와 같이 설정하세요.
예시) http://yourwebsite/blog 가 웹사이트 주소일 경우

  • url : http://yourwebsite/blog
  • root: /blog/

3. Directory

속성 설명
source_dir 소스폴더, 컨텐츠가 저장되는 폴더
public_dir 공개폴더, 생성된 static 파일이 저장되는 폴더
tag_dir 태그 폴더, public 이하에 있습니다.
archive_dir 아카이브 디렉토리
category_dir 카테고리 디렉토리
skip_render 렌더링 되지 않고 raw상태로 public 폴더에 이동될 경로

4. Writing

속성 설명
new_post_name 새 포스트의 파일 이름 형식
default_layout 기본 레이아웃
titlecase 제목을 titlecase로 변환할지 여부
filename_case 1 은 소문자로, 2는 대문자로 변환.
render_drafts 드래프트 렌더링 여부
post_asset_folder Asset Folder 활성화 여부
relative_link root 폴더를 기준으로 상대경로 생성 여부
highlight 코드블럭 설정

5. Category & Tag

속성 설명
default_category 기본 카테고리 설정
category_map 카테고리 slug
tag_map 태그 slug

6. Date / Time format

Hexo는 시간정보를 처리하기 위해 Moment.js 를 사용합니다.

속성 설명
date_format 날짜 형식
time_format 시간 형식

7. Pagination

속성 설명
per_page 한 페이지에 나타날 포스트의 양
pagination_dir 페이지네이션 디렉토리
